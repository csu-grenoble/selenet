
# First Local Deployment Launch

- npm install: install all dependencies if they are not already installed (check that the node_modules folder exists).
- npm run build: build from source files.
- Go to: https://ion.cesium.com/
- Create an account if you do not already have one.
- Open the Access Token section, create a token if you do not already have one, and copy it.
- To start the application: API_URL=my_token npm start

Steps 3 to 5 only need to be done the first time. After that, you can reuse the same token to run the application.

Once you are in cesium Ion, you need to add the moon asset to your assets. 
Go to **Asset Depot**, and type "Moon" in the search bar. Add the **Cesium Moon** asset to your asses by clucking the "+"

It should then show up in the **My Assets** Tab, with ID **2684829**
