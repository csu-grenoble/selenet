# Destination 🌑🌒🌔🌕 Moon 🌕🌖🌘🌑 - SeleNet 

SeleNet is a simulation platform for lunar IoT networks that uses SPICE-based orbital physics and a CesiumJS 3D interface to visualize satellites, ground devices, and communication links. It will help students and teachers to model and evaluate the performance of lunar satellite constellations.
